## Architecture

Three-layer strategy:
  1. Router handles commands deterministically (14/20 turns, zero LLM cost)
  2. LLM config constrains output (stop tokens, repeat penalty, budget limits)
  3. J handles judgment only (identity, math, recall, summary)

## Commands

quit / exit      — End session
/help            — Show all commands
/tools           — List available tools
/plan <goal>     — Agent mode: plan → execute → verify (buffer-based at ≤2048)
/steps <steps>   — Manual step injection (numbered or tool commands)
/buffer          — Show task buffer state
/buffer clear    — Clear task buffer
/model <name>    — Hot-swap model (e.g. /model qwen2.5-coder:14b)
/mode <level>    — Change autonomy (manual/semi/auto-safe/auto-full)
/memory          — Show working memory
/reflect         — Manually compress working memory
/integrity       — Check file integrity
/refactor        — Cross-file AST analysis
/optimize [path] — Five Masters code optimizer
/report          — Generate HTML task report
/sandbox         — Run pre-push validation
