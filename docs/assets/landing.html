Plan → Execute → Verify

DAG-based task planner decomposes goals into dependency-aware steps. Independent steps run in parallel. Each step is verified before continuing.

3-Tier Memory

Active context reconstructed every turn. Rolling working memory with BM25 retrieval. Persistent long-term memory across sessions. Weight-triggered auto-pruning.

14+ Dev Tools

File I/O, bash, git, search, tree, test runner, SQL, integrity hashing, scaffolding — all auto-discovered from tools/run/.

🔨

Inference Tool Forge

"Build a tool for X" → J researches the domain, generates code, validates in sandbox, and hot-registers — mid-session, no restart.

Self-Healing Circuit Breaker

Detects stuck loops, repeat errors, runaway turns. Graduated response: redirect → new approach → force skip. Never wastes your tokens spinning.

Pre-Push Sandbox

5-check validation gauntlet: conflict markers, py_compile, AST parse, tests, Five Masters code quality. No broken code leaves the drive.

🔍

AST Code Intelligence

Multi-file analysis: circular imports, unused imports, dead code, variable shadows, rename helpers. Generates dark-theme HTML reports.

Streaming Output

See builds, tests, and processes in real-time. Line-by-line streaming with 64 KB capture cap. No waiting for long ops to finish.

Dual Backend
